# Disagreements

Append entries here when an auditor output is wrong — false positive, missed
flag, wrong category, or just unclear. These are the raw signal for the next
round of prompt tuning and eval-fixture additions.

Format:

```
## YYYY-MM-DD [case brief]

Auditor said: [quote]
Reality: [what actually happened / why it was wrong]
Tag: [false-positive | missed-flag | wrong-category | unclear]
```
